The MatriX Bot

Hello! Welcome to The MatriX. I'm The MatriX Bot, will update you for all matches results. You can message here, which would be readable for all connected user to this web app.


How to play

- Search player via 'Search' tab

- You'll be connected to a random player

- On connection his/her username will be shown on third tab

- Now Start filling the cells of the matrix (Your Color: Blue)

- Whichever player would covered more cells than opponent will be the winner

- You also will be notified if opponent has quit the game in between via third tab, so you can again search for new random player

- You can message via 'Chat' tab, which is common chat group for all players

- Let's play, have fun!
