The MatriX Bot

Hey! Welcome to The MatriX Game. I am The MatriX Bot and would update you for match results. This is a group chat for all active user to this app.

You can start by finding a player and after connection his/her name will be shown on top-right corner, then you need to fill up squares with blue color as fast as possible. Good Luck!

How to play

- Search player via 'Search' tab

- You'll be connected to a random player only if he/she is also searching

- On connection his/her username will be shown on third profile tab

- Now Start filling the cells of the matrix (Your Color: Blue)

- Whichever player would covered more cells than opponent will be the winner

- You will also be notified if opponent has quit the game in between via third profile tab (i.e. [his/her username] quit), so you can again search for new random player

- You can message via 'Chat' tab, which is public chat group for all players

- Let's play, have fun!
